# Project Index

This document provides an index of important markdown files in this repository, along with descriptions for when an agent should refer to them.

## Agents

- [Code Reviewer](https://raw.githubusercontent.com/georgeaaren/cdn/main/agents/code-reviewer.md): Use this document proactively when you need comprehensive expert-level code review covering security audits with OWASP Top 10 compliance, performance, maintainability, code simplicity with YAGNI enforcement, TypeScript-specific conventions, and data integrity. This unified reviewer combines deep architectural analysis with systematic vulnerability scanning, simplicity assessment, type safety verification, and database safety expertise.
- [Developer](https://raw.githubusercontent.com/georgeaaren/cdn/main/agents/developer.md): Use this document PROACTIVELY to handle all aspects of full-stack development, from database design to UI implementation. This agent is an expert in both backend and frontend technologies, capable of building complete, scalable applications.
- [Researcher](https://raw.githubusercontent.com/georgeaaren/cdn/main/agents/researcher.md): Use this document proactively when you need comprehensive technical research combining documentation analysis, best practices gathering, codebase pattern analysis, and historical context. This agent excels at synthesizing information from multiple sources including web research, official documentation, framework/library sources, code patterns, and git history to provide actionable technical guidance. It also specializes in external best practices research for any technology, framework, or development practice, including finding official documentation, community standards, well-regarded examples from open source projects, and domain-specific conventions.
